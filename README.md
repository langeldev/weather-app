### Local installation

```sh
git clone https://github.com/langeldev/weather-app.git

cd weather-app

npm install
```

#### Api key
  * Create API KEY [openweathermap](https://openweathermap.org/)

  * update.env.example:  **VITE_API_KEY**=*YOUR_API_KEY*
```sh
npm run dev
```
